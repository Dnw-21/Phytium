## 配置文件说明

`config.json` 中可调参数：

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `refresh_interval_ms` | 1000 | 前端轮询周期 (ms)，1 step = 1 秒 |
| `log_buffer_size` | 200 | 本地日志最大条数（每节点独立） |
| `heartbeat_timeout_ms` | 3000 | 心跳超时 (ms) |
| `heartbeat_source` | "mock" | 心跳源 (mock/lora) |
| `heartbeat_simulated` | {"node_1": true, "node_2": true, "node_3": true} | Mock 心跳模拟 |
| `wifi.ssid` / `wifi.password` | "C2-218" / "12345678" | WiFi 配置 |
| `weather.enabled` | true | 天气服务开关 |
| `notify.feishu_enabled` | true | 飞书推送开关 |
| `notify.wechat_enabled` | true | 微信推送开关 |
| `nodes[].enabled` | [true, true, true] | 三节点全部启用 |
